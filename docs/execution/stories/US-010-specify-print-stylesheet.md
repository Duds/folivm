# US-010 — Specify print stylesheet

**As an** author,  
**I want to** specify a print stylesheet (CSS)  
**so that** preview and export use my styles.

## Acceptance Criteria

- [ ] I can set the path to a print stylesheet (project-relative or absolute)
- [ ] The stylesheet is applied to the preview
- [ ] The stylesheet is used for PDF export (EP-004)
- [ ] Changing the stylesheet updates preview and future exports
