# US-080 — Switch to structural mode

**As an** author,  
**I want to** switch to structural mode from the toolbar or keyboard  
**so that** I can work on the document structure.

## Acceptance Criteria

- [ ] I can switch to structural mode from a toolbar button (mode switcher)
- [ ] I can switch to structural mode via a keyboard shortcut
- [ ] When I switch, the editor view changes to show the heading hierarchy
- [ ] The current document mode is indicated in the mode switcher (segmented control)
