# US-033 — Choose export path

**As an** author,  
**I want to** choose where exports go (default deliverables/)  
**so that** I control where files are written.

## Acceptance Criteria

- [ ] PDF and DOCX export default to deliverables/ within the project
- [ ] I can override the export path for a given export (e.g. save-as dialog)
- [ ] Export path is within the project or user-chosen location
- [ ] File naming preserves or derives from document title/filename
