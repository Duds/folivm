# US-102 — Extension contract for sidebar/panel registration

**As a** developer,  
**I want** a clear contract for how extensions register sidebar icons and panel content  
**so that** the AI assistant (and future extensions) can plug into the right panel.

## Acceptance Criteria

- [ ] Extension manifest or API defines: icon, label, panel slot (e.g. right panel)
- [ ] Core app renders extension panel when extension is installed and enabled
- [ ] Contract documented for extension authors
