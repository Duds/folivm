# US-040 — Request LLM assistance

**As an** author,  
**I want to** request LLM assistance with my document  
**so that** I can draft or refine content.

## Acceptance Criteria

- [x] I can trigger an LLM request (toolbar button, menu, or shortcut)
- [x] Each request is user-initiated; no automatic generation
- [x] I can provide a prompt or instruction
- [x] The current document content is sent with the request
- [x] Response is displayed as a suggestion; document is unchanged until I accept
