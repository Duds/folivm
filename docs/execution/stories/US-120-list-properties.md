# US-120 — List properties (bullet style, number format)

**As an** author,  
**I want** to edit list properties (bullet style, number format) in the properties panel when a list is selected  
**so that** I can change list appearance without editing Markdown syntax directly.

## Acceptance Criteria

- [ ] When a list is selected in the structure tree or editor, properties panel shows list controls
- [ ] Bullet style: disc, circle, square (or semantic equivalents)
- [ ] Number format: 1, 2, 3 / 1.1.1 / a, b, c / i, ii, iii
- [ ] Multi-level numbering for nested lists
