# US-114 — Toggle non-visible / non-printing characters

**As an** author,  
**I want** to toggle visibility of non-printing characters (paragraphs, spaces, tabs, etc.)  
**so that** I can inspect document structure.

## Acceptance Criteria

- [x] View menu item "Show non-printing characters" with shortcut ⌘⇧8
- [x] When enabled: spaces (·), tabs (→), hard breaks (↵) visible as overlay symbols
- [x] Preference persisted in localStorage
