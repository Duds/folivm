# US-021 — Open project and see documents

**As an** author,  
**I want to** open a project and see my documents  
**so that** I can work on them.

## Acceptance Criteria

- [ ] I can open an existing project by selecting its root folder
- [ ] I see a list or tree of documents in working/ and deliverables/
- [ ] I can open a document from the list; it loads in the editor
- [ ] Documents are shown with readable names (filename or title)
