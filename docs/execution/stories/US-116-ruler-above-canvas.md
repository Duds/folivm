# US-116 — Ruler above document canvas

**As an** author,  
**I want** a horizontal ruler above the document  
**so that** I can visualise margins and scale.

## Acceptance Criteria

- [x] Ruler appears above document when in document view
- [x] Ruler shows scale in mm, cm, or inch (configurable in Settings)
- [x] Ruler shows margin indicators; default mm per Australian conventions
- [x] Ruler hidden in Preview and Outline views
