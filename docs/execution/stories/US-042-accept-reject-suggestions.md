# US-042 — Accept or reject AI suggestions

**As an** author,  
**I want to** accept or reject AI suggestions before they enter the document  
**so that** I stay in control of the content.

## Acceptance Criteria

- [x] AI-generated content is shown in a suggestion UI (sidebar, modal, or inline)
- [x] I can accept the suggestion; content is inserted at the chosen position
- [x] I can reject the suggestion; document remains unchanged
- [x] I can edit the suggestion before accepting
- [x] No content enters the document without explicit accept
