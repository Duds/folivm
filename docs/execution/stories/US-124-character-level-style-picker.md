# US-124 — Character-level style picker (emphasis, strong, code, link)

**As an** author,  
**I want** to apply character-level styles (emphasis, strong, inline code, link) via the right panel style picker  
**so that** I can format text runs without relying on the context menu.

## Acceptance Criteria

- [ ] When a text range (collapsed or multi-character) is selected, the properties panel shows character-style controls
- [ ] Character style picker offers toggles for: emphasis (italic), strong (bold), inline code, link
- [ ] Toggling a character style applies or removes the corresponding Markdown formatting
- [ ] Custom character styles (e.g. Label, Term) deferred to Phase 2
