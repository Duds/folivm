# US-119 — Context-sensitive properties for headings, paragraphs, lists, blocks

**As an** author,  
**I want** the properties panel to show controls relevant to the selected node  
**so that** I can edit heading, paragraph, list, and block properties without hunting for controls.

## Acceptance Criteria

- [x] Document node: title, created, updated, author; page size, orientation; heading numbering
- [x] Heading node: level (read-only), font size (VariablePicker), font weight — VariablePicker/weight not wired
- [ ] Paragraph: alignment, indent, line height — not implemented
- [ ] List: bullet style, number format, multi-level numbering — not implemented
- [ ] Semantic blocks: block type, colours — not implemented
