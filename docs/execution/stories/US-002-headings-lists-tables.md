# US-002 — Add headings, lists, and tables

**As an** author,  
**I want to** add headings, lists, and tables  
**so that** I can structure my content.

## Acceptance Criteria

- [ ] I can add headings (H1–H6) via toolbar or shortcut; they render and save as Markdown
- [ ] I can add ordered and unordered lists; nesting works
- [ ] I can add tables with headers and body
- [ ] Saved output is valid Pandoc Markdown (e.g. `# Heading`, `- item`, `| col |`)
- [ ] Round-trip: reload preserves structure
