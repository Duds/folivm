# US-041 — LLM receives brief and context files

**As an** author,  
**I want** the LLM to receive my brief and selected context files  
**so that** it has project context when assisting.

## Acceptance Criteria

- [x] I can select which project files (brief, inputs/, context/) to include with each request
- [x] Selected files are read and sent with the LLM request payload
- [x] Brief or primary context path is configurable or discoverable
- [x] Context is included without manual copy-paste
