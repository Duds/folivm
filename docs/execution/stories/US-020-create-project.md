# US-020 — Create new project

**As an** author,  
**I want to** create a new project  
**so that** I get the standard folder schema (inputs/, working/, context/, deliverables/).

## Acceptance Criteria

- [ ] I can create a new project from the app (menu or welcome screen)
- [ ] I choose or specify the project root path
- [ ] The system creates inputs/, working/, context/, deliverables/ if they do not exist
- [ ] Project is opened and ready for document creation
- [ ] Optional Folivm.yaml can override schema (future; Phase 0 may use default only)
