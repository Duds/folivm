# US-099 — Document tab bar UI

**As an** author,  
**I want** to open multiple documents in tabs with a tab bar above the editor  
**so that** I can switch between documents without losing context.

## Acceptance Criteria

- [ ] Tab bar appears in editor canvas area; each open document has a tab
- [ ] Tab label shows filename or short path
- [ ] Active tab is visually distinct; clicking a tab switches to that document
- [ ] Tab close button (or ⌘W) closes the tab
