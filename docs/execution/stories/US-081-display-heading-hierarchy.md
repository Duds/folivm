# US-081 — Display heading hierarchy in structural mode

**As an** author,  
**I want** structural mode to display H1–H4 as a collapsible, indented hierarchy with body text hidden  
**so that** I see the document spine.

## Acceptance Criteria

- [ ] Structural mode displays all H1–H4 headings from the document
- [ ] Headings are shown as an indented hierarchy (H1 least indented, H4 most)
- [ ] Body text (prose, lists, tables, fenced divs) is hidden or summarised
- [ ] The hierarchy reflects the actual document order
- [ ] Empty headings or placeholder text are shown if a heading has no content
