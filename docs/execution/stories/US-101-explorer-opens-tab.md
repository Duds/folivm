# US-101 — Explorer click opens or activates tab

**As an** author,  
**I want** clicking a document in the explorer to open it in a new tab or switch to its existing tab  
**so that** I never have duplicate tabs for the same document.

## Acceptance Criteria

- [ ] Clicking a document in explorer: if already open, switch to that tab; else add new tab and activate
- [ ] No duplicate tabs for same document path
