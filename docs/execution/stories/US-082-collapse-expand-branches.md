# US-082 — Collapse and expand heading branches

**As an** author,  
**I want to** collapse and expand heading branches in structural mode  
**so that** I can focus on parts of the hierarchy.

## Acceptance Criteria

- [ ] Each heading with children has a collapse/expand control
- [ ] I can collapse a branch to hide its children
- [ ] I can expand a branch to show its children
- [ ] Collapsed state is visual only; it does not alter the document
- [ ] Collapse/expand state may persist during the session (implementation detail)
