# US-023 — Add brief and context files

**As an** author,  
**I want to** add a brief or context files to my project  
**so that** the LLM can use them when assisting.

## Acceptance Criteria

- [ ] I can place files in context/ (e.g. brief.md) or inputs/
- [ ] When requesting LLM assistance (EP-005), I can select which context files to include
- [ ] Selected files are sent with the LLM request
- [ ] Brief or primary context is discoverable (e.g. configurable path or convention)
