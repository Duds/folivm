# US-030 — Export to PDF

**As an** author,  
**I want to** export the current document to PDF  
**so that** I can deliver it to my client.

## Acceptance Criteria

- [ ] I can trigger PDF export (toolbar or menu)
- [ ] Export uses the author-configured print stylesheet
- [ ] Output is written to deliverables/ or a path I choose
- [ ] Export completes in under 30 seconds for documents up to 50 pages
- [ ] PDF is professional quality without manual post-processing
