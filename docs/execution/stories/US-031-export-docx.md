# US-031 — Export to DOCX

**As an** author,  
**I want to** export the current document to DOCX  
**so that** I can deliver it in Word format.

## Acceptance Criteria

- [ ] I can trigger DOCX export (toolbar or menu)
- [ ] Export uses optional reference DOCX template if configured
- [ ] Output is written to deliverables/ or a path I choose
- [ ] DOCX is professional quality without manual post-processing
