# US-104 — Core app: extension slot, no built-in AI when extension absent

**As an** author,  
**I want** core Folivm to have no AI UI when the AI extension is not installed  
**so that** I get a lean app if I don't need LLM assistance.

## Acceptance Criteria

- [ ] Without AI extension: no LLM panel, no LLM config, no AI-related menu items
- [ ] Right panel shows extension slot; when no extension registered, slot is empty or shows "Install extensions"
- [ ] With AI extension installed: assistant panel appears as today
