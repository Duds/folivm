# US-012 — Hierarchical heading numbering

**As an** author,  
**I want to** have hierarchical heading numbering (1, 1.1, 2.2.1)  
**so that** my document has numbered sections.

## Acceptance Criteria

- [ ] When enabled in the stylesheet, headings display with CSS counter numbering
- [ ] H1, H2, H3 produce 1, 1.1, 2.2.1 style numbering
- [ ] Numbering appears in preview and exported PDF
- [ ] Numbering is controlled by CSS (no hardcoded logic in document)
