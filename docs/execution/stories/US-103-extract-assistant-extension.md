# US-103 — Extract assistant panel into AI extension package

**As a** developer,  
**I want** the LLM assistant UI and logic extracted into an extension package  
**so that** core Folivm can run without it.

## Acceptance Criteria

- [ ] AI extension package contains: assistant panel UI, provider config, request/response logic
- [ ] Extension stores API key in Folivm.yaml or extension-specific config
- [ ] Extension registers as right-panel view when installed
