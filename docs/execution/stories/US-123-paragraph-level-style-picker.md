# US-123 — Paragraph-level style picker (heading, body, block type)

**As an** author,  
**I want** to apply paragraph-level styles (heading level, body, block type) via the right panel style picker  
**so that** I can change block appearance using Folivm variables without the context menu.

## Acceptance Criteria

- [ ] When a heading, paragraph, or semantic block is selected, the properties panel shows paragraph-style options
- [ ] Paragraph style picker offers: heading level (H1–H4), body, block type (callout, executive-summary)
- [ ] Options are bound to Folivm variables (e.g. `--text-doc-h1`, `--color-block-callout-*`)
