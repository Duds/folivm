# US-011 — Print preview

**As an** author,  
**I want to** see a print preview that matches my stylesheet  
**so that** I know how export will look before exporting.

## Acceptance Criteria

- [ ] I can toggle or view a print preview pane
- [ ] Preview applies the configured stylesheet
- [ ] Headings, lists, tables, blockquotes, and fenced divs render with correct structure
- [ ] Preview approximates final PDF appearance (structural match)
- [ ] Preview updates when I change content or stylesheet
