# US-097 — Empty canvas layout and Folivm watermark

**As an** author,  
**I want** the empty editor canvas to show no page chrome and a Folivm watermark  
**so that** the experience is clean and branded when no document is open.

## Acceptance Criteria

- [ ] When no document is open: no page outline, shadow, or paper container
- [ ] Folivm watermark: large, low-opacity wordmark or logo, centred
- [ ] Layout is aesthetically balanced
